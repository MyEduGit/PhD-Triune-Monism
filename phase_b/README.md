# Claw Fleet — Phase B dashboard & pipeline

One Streamlit app that does three jobs:

1. **Awesome Foursome dispatch** — ChatGPT, Claude, Grok, Gemma4 buttons
   plus a Council-of-Seven webhook button. Real HTTP calls. Missing API
   keys are reported honestly, not faked.
2. **Phase B SRE drill-down** — filterable conflicts & hypotheses from
   `json_repository/phase_b_sre_report.json`. Filters by model, subject
   (predicate filter on the roadmap). Drill-down shows extracted triple
   + raw evidence.
3. **Zero-touch orchestrator** — one button runs `launcher.sh`, which
   walks EE → CM/HG → visualizer → email/Telegram notifications.

## Files

```
phase_b/
├── app.py               # Streamlit unified dashboard
├── launcher.sh          # zero-touch pipeline (EE → CM/HG → viz → notify)
├── scheduler.sh         # install/remove weekly Friday cron
├── requirements.txt     # streamlit, requests
├── assets/              # dashboard Markdown outputs (created at runtime)
├── json_repository/     # EE outputs + phase_b_sre_report.json
└── scripts/             # backend logic modules (populate these):
    ├── ee_extract_triples.py
    ├── sre_process_and_generate.py
    └── sre_visualizer.py
```

## First run

```bash
cd phase_b
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# (optional) export keys so the Foursome buttons can call remote APIs
export ANTHROPIC_API_KEY=sk-ant-...
export OPENAI_API_KEY=sk-...
export XAI_API_KEY=xai-...

# (optional) notifications
export EMAIL_TO=you@example.com
export TELEGRAM_BOT_TOKEN=...
export TELEGRAM_CHAT_ID=...

streamlit run app.py
```

Open http://localhost:8501.

## Run the pipeline from the command line

```bash
bash launcher.sh                     # pipeline only
bash launcher.sh --ui                # pipeline + launch Streamlit
bash launcher.sh --skip-notify       # no email/telegram
```

## Schedule weekly Friday 09:00 run

```bash
bash scheduler.sh install     # add cron entry
bash scheduler.sh status      # confirm
bash scheduler.sh remove      # uninstall
```
