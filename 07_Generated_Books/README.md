# 07 — Generated Books

This folder is the PhD-side **destination vault** for books generated by
the URANTiOS BookWriter (`URANTiOS/bookwriter/`).

- Every book arrives as its own subfolder under `Books/`.
- The cover file (`00 - <Title>.md`) is an Obsidian MOC for the book.
- The master index (`Books/_Books MOC.md`) lists every generated book.
- Every in-text citation `(120:4.5)` is linkified to `[[UB-120-04-005|120:4.5]]`
  so it lights up a Urantia-Book-per-paragraph vault with backlinks.

## Writing a new book here

From the `URANTiOS` repo:

```bash
bookwriter write \
  --theme "The Bestowal Career of Michael of Nebadon" \
  --chapters 12 \
  --vault /home/user/PhD-Triune-Monism/07_Generated_Books
```

Or from Python:

```python
from bookwriter import BookWriter, Corpus, ObsidianVault

corpus = Corpus.load()
vault  = ObsidianVault(root="/home/user/PhD-Triune-Monism/07_Generated_Books")
writer = BookWriter(corpus=corpus, vault=vault)
writer.write(theme="The Three Absolutes", chapters=10)
```

## Governance

Every book passes the **Lucifer Test** before it lands here.
Every chapter's YAML frontmatter records:

- The book it belongs to
- The chapter number and thesis
- The list of Urantia Book citations it contains
- The Lucifer verdict (`PASS` / `REVISE` / `FAIL`)
- The number of revision passes applied

Books without a `PASS` verdict should be reviewed before incorporation
into the PhD dissertation or external publication.
